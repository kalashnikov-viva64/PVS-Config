Командный файл PVS-Analyse.bat предназначен для проведения статического анализа кода с помощью PVS-Studio.

Использование: PVS-Analyse.bat <Platform> <DelayMinutes>
<Platform> - платформа x86 или x64
<DelayMinutes> - время анализа в минутах

Чтобы повысить надежность работы статического анализа был использован ряд приемов:
 * 3 попытки анализа,
 * контроль создания отчета *.plog,
 * принудительное завершение работы Visual Studio по истечению таймаута,
 * удаление временных файлов Visual Studio.
